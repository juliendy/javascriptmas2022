# 🤖 Insert Dashes

## Challange
Transform a given sentence into a new one with dashes between each two consecutive letters.

Example

For inputString = “aba caba”, the output should be insertDashes(inputString) = ‘a-b-a c-a-b-a’.

Hints:

1. join()
2. split()
